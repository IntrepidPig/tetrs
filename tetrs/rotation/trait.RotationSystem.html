The RotationSystem trait is implmented by all rotation systems.

When implementing RotationSystem, the only thing that is required to implement are the offset values for each of the main blocks I, J, L, S, Z, T, O.

See the srs.rs file for an example.

Required Methods

fn data(&self, ty: Type, rotation: Rotation) -> &'static [(usize, usize)]

Returns a static array of offset values for the specified Type and Rotation.

fn minp(&self, id: Type, r: Rotation) -> (usize, usize)

Returns the minimum offset of the first piece in a block.

Return the offset from the (x, y) bounding coordinate to the first non-empty piece in a block. This row by row from y = 0 onwards.
