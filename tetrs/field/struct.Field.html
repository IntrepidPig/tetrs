A Field is simply a 2-D Vec with some corresponding options.

Internally we use a Vec of Vec's for easier resizing, but all rows must be of equal length, so it can be thought of as rectangular for all purposes.

Note

If one wants to only iterate over the viewable portion of the array, this can be done with the following bounds:

for y in self.hidden..self.height {
    ...
}

instead of the more presumed

for y in 0..self.height {
}

This is because the height field includes the specified hidden portion.

Fields

The height of the field.

The height of the hidden region of the field.

This is a subregion of height, and does not actually add any more height to the field.

The initial spawn of a Block on this field.

fn occupies(&self, (x, y): (usize, usize)) -> bool

Return true if the value at the specified location is non-empty.

This is a convenience function which queries at and checks if the result is of empty Id.

If the input values are outside the range of the field then true is returned. Explicit bounds checking is left up the user.
