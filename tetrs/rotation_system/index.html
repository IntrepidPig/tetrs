Module tetrs::rotation_system [−] [src]

This modules provides an interface for dealing with different block offsets.

Offsets refer to particular rotation specifications. For example, the SRS and Akira style rotation systems which each contain different offset values which can both be used if they implement the RotationSystem trait.
