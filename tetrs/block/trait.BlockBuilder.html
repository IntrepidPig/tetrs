Traits for building a block.

This is used to override given default block values to more specific values if required.

These should only be used on construction of a new block with block::new(). This is not enforced, but limited checking is done internally and the results may not be as one would expect.

No collision testing is done on the resulting block, so it is up to the caller to perform this.

Examples

use tetrs::block::{Type, Rotation, Block, BlockBuilder};

let block = Block::new(Type::I)
                  .set_rotation(Rotation::R270);
                  .set_position((5, 10))
                  .set_rotation_system(rotation::SRS::new());

See block::new() for what default values are used.
