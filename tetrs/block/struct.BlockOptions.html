Optional values which can be set when initializing a Block.

The default values are:

BlockOptions {
    x: None,
    y: None,
    rotation: Rotation::R0,
    rotation_system: rotation_system::SRS
}

If an x or y value is Some(..) then it will override the fields spawn position, which is usually used on Block construction.

Examples

use tetrs::import::*;

// Has x: None, y: None, rotation: Rotation::R0, rotation_system: "dtet"
let options = BlockOptions {
    rotation_system: rotation_system::new("dtet").unwrap(),
    ..Default::default()
};
